🧠 About  
Most frontend projects stop at “it works.” This one focuses on how it *feels*. Built to explore high-end UI engineering using animations, layout transitions, and responsive design to deliver a smooth, polished experience.

⚡ Highlights  
- Animation-first approach using GSAP  
- Smooth performance with complex animations  
- Micro-interactions and precise timing  
- Clean, scalable component architecture  

🛠 Tech Stack  
React.js • Tailwind CSS • GSAP  

🔥 Features  
Scroll-triggered animations, clip-path transitions, 3D hover effects, video-integrated sections, fully responsive design  

📈 What this shows  
Strong frontend fundamentals, ability to build premium UI, understanding of performance vs animation trade-offs, attention to detail  

🧠 Learnings  
Animations need optimization, small UI details matter, structure is key for complex UI, smooth UX depends on timing
